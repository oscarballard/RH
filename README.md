# RH

Es un proyecto el cual se encarga de la gestion de candidatos para recursos humano. Este proyecto esta dividido en dos proyecto el cual es la app que seria el cliente del proyecto en donde los usuarios podran consumir los datos y registrar los mismo. La otra parte del proyecto es una apirest la cual tiene toda la logica del negocio.
